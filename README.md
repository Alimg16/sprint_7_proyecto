# sprint_7_proyecto
Proyecto del sprint 7 

Descripción

Esta aplicación web permite realizar una exploración básica de un conjunto de datos de anuncios de venta de vehículos usados en Estados Unidos.

La aplicación fue desarrollada con Streamlit, Pandas y Plotly como parte del Sprint 7 del programa de Ciencia de Datos de TripleTen.

Funcionalidades
* Visualizar un histograma de la distribución del kilometraje (odometer) de los vehículos.
* Visualizar un gráfico de dispersión que muestra la relación entre el kilometraje (odometer) y el precio (price) de los vehículos.
* Interactuar con los gráficos mediante Plotly.

Tecnologías utilizadas
* Python
* Pandas
* Plotly
* Streamlit

Cómo ejecutar la aplicación
1. Clonar este repositorio:
git clone https://github.com/Alimg16/sprint_7_proyecto.git
2. Entrar al directorio del proyecto:
cd sprint_7_proyecto
3. Instalar las dependencias:
pip install -r requirements.txt
Ejecutar la aplicación:
streamlit run app.py

La aplicación esta disponible en:
https://vehicle-analysis-f9jr.onrender.com/
